# VF-T20F256 ADC UI 项目模块与功能
| 代码文件名               | 核心功能                                                                 |
|--------------------------|--------------------------------------------------------------------------|
| `adc_ui_top.v`           | 项目顶层整合模块，将ADC采集、LCD显示、SDRAM存储、PLL时钟等子模块连接起来，实现各部分的信号交互，让整个系统正常运行 |
| `ui_display.v`           | 负责LCD屏幕的显示控制，生成800×480分辨率所需的行同步、场同步、数据使能等时序信号，同时在屏幕上绘制ADC采集到的波形和参数文本 |
| `adc_data_sync.v`        | 解决ADC与系统时钟不同步的问题，将ADC的40MHz时钟下的数据，平稳转换到系统25MHz时钟下，避免数据传输时出现错误 |
| `sdram_ctrl_wrapper.v`   | 封装SDRAM控制器IP，使其能适配开发板上的64Mbit SDRAM，实现对SDRAM的读写操作 |
| `pll_lcd.v`              | 将开发板上25MHz的晶振时钟，倍频到LCD屏幕所需的33.3MHz像素时钟，为LCD显示提供稳定的时钟源 |
| `char_generator.v`       | 在LCD屏幕上生成“ADC Value: XXX”的文字内容，其中“XXX”会实时替换为ADC的采集数值 |
| `wave_ctrl.v`            | 实现ADC波形在Y轴方向的缩放功能（缩放范围0.5倍到2倍），支持通过RISC-V接口设置缩放系数 |
| `touch_driver.v`（可选） | 触摸屏的I2C驱动模块，能读取触摸屏的触摸坐标，且坐标适配LCD的800×480分辨率 |
